# Finance Controller — Vision

## Problem

Managing personal finances manually is error-prone and time-consuming. Spreadsheets lack structure, and most apps don't offer the flexibility needed for custom categorization and visualization.

## Objectives

- Track income, expenses, and transfers across multiple accounts
- Categorize transactions with customizable, hierarchical categories (with icons/colors)
- Visualize financial health through customizable dashboards with drag-and-drop widgets
- Support recurring transactions (salary, rent, subscriptions)
- Serve as a portfolio piece demonstrating production-quality fullstack development
- Be extensible as a SaaS/freelance base (multi-tenant architecture from day one)

## Non-objectives

- Open Finance / bank API integration (manual entry + import only)
- Mobile native app (responsive web is sufficient)
- AI-powered insights (future consideration)

## Personas

- **Primary**: Solo developer managing personal finances
- **Secondary**: Potential freelance clients needing a white-label finance tool

## Success Metrics

- Register and create first transaction in under 5 minutes
- Dashboard renders key financial overview at a glance
- Clean, documented codebase suitable for portfolio presentation
